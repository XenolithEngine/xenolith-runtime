== Internationalized Domain Names (IDN)

The `sprt::idn` namespace provides utilities for handling internationalized domain names (IDN), including Punycode encoding/decoding and TLD validation.

=== Punycode Encoding/Decoding

Punycode is a character encoding used to represent Unicode strings as ASCII strings, which is necessary for domain names in DNS.

==== puny_encode

Encodes a UTF-8 string in Punycode using an internal temporary buffer.

*Signature*
[source,cpp]
----
bool puny_encode(const callback<void(StringView)> &cb, StringView source, bool makeUrlPrefix);
----

*Parameters*
* `cb` (callback<void(StringView)> &) - Callback to receive the encoded result
* `source` (StringView) - The UTF-8 string to encode
* `makeUrlPrefix` (bool) - If true, prefixes the result with "xn--"

*Return Value*
- Returns `true` on success, `false` on failure

*Example*
[source,cpp]
----
sprt::idn::puny_encode([](sprt::StringView result) {
    // result contains the Punycode-encoded string
}, sprt::StringView{u8"фр.язык"}, true);
----

==== puny_decode

Decodes a Punycode string to UTF-8 using an internal temporary buffer.

*Signature*
[source,cpp]
----
bool puny_decode(const callback<void(StringView)> &cb, StringView source, bool prefixed);
----

*Parameters*
* `cb` (callback<void(StringView)> &) - Callback to receive the decoded result
* `source` (StringView) - The Punycode string to decode
* `prefixed` (bool) - If true, checks for "xn--" prefix and removes it before decoding

*Return Value*
- Returns `true` on success, `false` on failure

*Example*
[source,cpp]
----
sprt::idn::puny_decode([](sprt::StringView result) {
    // result contains the UTF-8 decoded string
}, sprt::StringView{u8"xn--fr-zya"}, true);
----

=== TLD Validation

==== is_known_tld

Checks whether a given top-level domain identifier is in the list of known domains.

*Signature*
[source,cpp]
----
bool is_known_tld(StringView);
----

*Parameters*
* `tld` (StringView) - The domain name to check (can be Punycode or UTF-8)

*Return Value*
- Returns `true` if the TLD is in the known list, `false` otherwise

*Note*
- The list is based on the official IANA TLD database at https://data.iana.org/TLD/tlds-alpha-by-domain.txt

*Example*
[source,cpp]
----
if (sprt::idn::is_known_tld(sprt::StringView{u8"xn--n3h.com"})) {
    // .com is a known TLD
}
----

=== Hostname Conversion

==== to_ascii

Converts a hostname from Unicode (UTF-8) to ASCII (Punycode).

*Signature*
[source,cpp]
----
bool to_ascii(const callback<void(StringView)> &, StringView source, bool validate);
----

*Parameters*
* `cb` (callback<void(StringView)> &) - Callback to receive the ASCII result
* `source` (StringView) - The UTF-8 hostname to convert
* `validate` (bool) - If true, validates that only valid characters are used

*Return Value*
- Returns `true` on success, `false` on failure

*Usage*
[source,cpp]
----
sprt::idn::to_ascii([](sprt::StringView ascii) {
    // ascii contains the Punycode-encoded hostname
}, sprt::StringView{u8"фр.язык"}, true);
----

==== to_unicode

Converts a hostname from ASCII (Punycode) to Unicode (UTF-8).

*Signature*
[source,cpp]
----
bool to_unicode(const callback<void(StringView)> &, StringView source, bool validate);
----

*Parameters*
* `cb` (callback<void(StringView)> &) - Callback to receive the Unicode result
* `source` (StringView) - The Punycode hostname to convert
* `validate` (bool) - If true, validates that only valid characters are used

*Return Value*
- Returns `true` on success, `false` on failure

*Usage*
[source,cpp]
----
sprt::idn::to_unicode([](sprt::StringView unicode) {
    // unicode contains the UTF-8 decoded hostname
}, sprt::StringView{u8"xn--n3h.com"}, true);
----

== Usage Scenarios

=== URL Handling

When working with internationalized domain names in URLs:

[source,cpp]
----
// Convert internationalized domain to ASCII for DNS
sprt::idn::to_ascii([](sprt::StringView ascii) {
    std::string url = "https://" + std::string(ascii);
    // url now contains the ASCII-encoded hostname
}, sprt::StringView{u8"фр.язык.com"}, true);
----

=== TLD Validation in URLs

[source,cpp]
----
sprt::idn::to_unicode([](sprt::StringView unicode) {
    if (sprt::idn::is_known_tld(unicode)) {
        // Valid internationalized domain
    }
}, sprt::StringView{u8"xn--n3h.com"}, true);
----

== Internal Buffers

All functions use internal temporary buffers for encoding/decoding operations. The callback-based API allows flexibility in how the results are handled.

*Note* - The callbacks receive `StringView` references to the results, which are valid as long as the operations complete successfully.
